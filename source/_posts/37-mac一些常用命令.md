- 同时按住 option(alt) 键，可以以列选中，类似于 sublime3中 按住 alt的列选中 。
```
Command + option(alt) 
```

- 剪贴板历史记录
```
Command + Shift + h
```

- 将文本内容复制到剪切板
```
pbcopy < a.txt
```
