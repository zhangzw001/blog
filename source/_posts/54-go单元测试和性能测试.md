###  首先这里有三个斐波那契的函数 fib.go
```go
func Fib(n int ) int {
 if n < 2 {
  return 1
 }
 return Fib(n-2) + Fib(n-1)
}


func Fib2(n int) int {
 var f = [3]int{0, 1, 1}
 for i := 2; i <= n; i++ {
  f[2] = f[0] + f[1]
  f[0] = f[1]
  f[1] = f[2]
 }
 return f[2]
}

func Fib3() func() int {
 var n,m int
 return func() int {
  if n <= 1 {
   n = 1
  }
  n,m = n+m,n
  return m
 }
}
```



###  先做一个单元测试 fib_test.go
```go
func TestFib(t *testing.T) {
 for i:= 0 ; i < 40 ; i++ {
  Fib(i)
 }
}

func TestFib2(t *testing.T) {
 for i:= 0 ; i < 90 ; i++ {
  Fib2(i)
 }
}

func TestFib3(t *testing.T) {
 f3 := Fib3()
 for i:= 0 ; i < 90 ; i++ {
  f3()
}
}
```

### go.mod使用说明

> 目录结构

```
├── fib
│   ├── fib.go
│   └── fib_test.go
├── main.go
└── go.mod
```

> main.go 
```go 
package main

import (
 "fib"
 "fmt"
)

func main() {
 for i :=0 ; i< 40; i++ {
  fmt.Printf("%d,",fib.Fib(i))
 }
 fmt.Println("")
 for i :=1 ; i< 90; i++ {
  fmt.Printf("%d,",fib.Fib2(i))
 }
 fmt.Println("")
 f3 := fib.Fib3()
 for i :=1 ; i< 90; i++ {
  fmt.Printf("%d,",f3())
 }
}
```

> 说明
```
1. go版本最好是1.13 (go1.11开始支持)
2. 首先 go mod init , 并修改go.mod
    增加: replace fib => ./fib
3. 然后进去到fib目录, 执行go mod init 
4. 在执行main.go 即可

```

### 一些常用命令
```
1. go test 	// 返回正确 或者返回错误详情
2. go test -v 	// 返回详情,包括正确的测试案例
3. go test -v -run="French|Can.*l"	// 执行指定的测试案例(参数-run对应一个正则表达式)
```
